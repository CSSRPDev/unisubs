filterBox is a flexible system to allow users to filter content.

When the filters are changed, we will make an AJAX request to the current URL. Respond using AJAXResponseRenderer to update the page.

If a sort has a default value, use data-[name]-default to specify it. Then if a user chooses that value, we won't display the filter.
